# 毕业设计 用户端

> 基于nodejs直播解决方案

## Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn dev

# build for production with minification
yarn build

# build for production and view the bundle analyzer report
yarn build --report
```

## OBS推流设置
```
url: rtmp://localhost/live

流名称: 填写申请的房间号
```
